# Power BI Dashboards

This project demonstrates business intelligence dashboards built using Power BI.

## Dashboards Included
- Revenue vs Actual vs Forecast
- Sold vs Redeemed analysis
- Sales performance dashboard
- Data quality and reconciliation report

## Key Features
- Data modelling (Star schema)
- DAX measures for KPIs
- Interactive filters and drill-down
